Bin Packer

 In the online MMO Elite Dangerous, there is a colonization system that allows the construction of stations in unclaimed systems. The construction of stations requires tens of thousands, or up to hundreds of thousands of materials.

 Finding the markets, planning each trip, and keeping track of the quantity required for each material felt tedious. In response, I created the first version of Bin Packer. The first version only took in a text file that matched a set format, and planned the materials to buy for each trip. However, that only solved the planning and the material quantity tracking; I still had to find each market.

 Each update aimed at automating a step that was previously done manually. I started by adding the ability to search the game's Journal files for the list of required materials. Then, I made it search online for markets that have the required materials. Finally, I made an algorithm to plan a route from the station, to the markets, and back to the station.
